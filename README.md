# Sim_de_activos_y_carteras_basados_en_pasivos
Proyecto que analiza la estrategia CPPI y su integración con LDI mediante simulaciones con modelos GBM y CIR. Evalúa protección de capital, estabilidad del funding ratio y desempeño en distintos escenarios de mercado. Incluye código de simulación y reporte técnico.
