# Sim_de_activos_y_carteras_basados_en_pasivos
Este proyecto implementa y analiza la estrategia CPPI (Constant Proportion Portfolio Insurance) y su integración dentro de un esquema Liability-Driven Investment (LDI) mediante simulaciones estocásticas con modelos GBM y CIR. A través de distintos escenarios de mercado, sectores y configuraciones de riesgo, se evalúa el comportamiento del portafolio, la protección del capital, la estabilidad del funding ratio y la efectividad de técnicas como duration matching y estrategias híbridas de protección. El repositorio incluye el código utilizado para las simulaciones y el reporte técnico completo del análisis.
